# Тизер-карта главной: мини-карточки + фикс перехода к большой карте

Дата: 2026-06-12. Одобрено Артёмом (мокап в чате; правка: карточки не должны перекрываться).

## Часть 1 — мини-карточки на карте-тизере (главная EN+RU)

**Что:** после того как точки легли (или сразу, если интро не играет), на тизере
появляются мини-карточки-поповеры у пинов: одна КВАРТИРА (у оранжевого лого-пина)
и одно МЕСТО из гида (у пина eat&drink/wellness). Фото + название + подпись
(flat · from ₾X / мир гида). Карточки кликабельны → страница квартиры / места.

**Данные:** настоящие, build-time из Supabase (apartments_public, guide_places,
только с фото). На каждый заход JS выбирает случайную пару из заготовленных
кандидатов (массив в data-атрибуте; SSR-рендер кандидата 0, img грузится только
выбранный — data-src + инлайн-скрипт).

**Раскладка без перекрытий (правка Артёма):** координаты пинов перестроены так,
чтобы у карточек были чистые зоны: карточка места над пином ~(52,30) занимает
y≈13–28%; карточка квартиры над лого-пином ~(72,62) занимает y≈45–60%. Зоны не
пересекаются между собой, с подписью (верх-лево) и кнопкой CTA (низ-право).
Прочие пины передвинуты из-под карточек.

**Breakpoints:** ≥880px — две карточки; ≤880px (тизер = квадрат) — одна
компактная карточка (квартира). Хвостик-указатель на пин сохраняется.

**Движение (§ безопасность движения):** проявление на месте — opacity + лёгкий
scale, без падения/удара. С интро: карточки появляются в finish() (после
посадки пинов), стаггер ~120мс. Без интро (повтор, мобилка, reduced-motion):
появляются через ~400мс после загрузки. Пока html.intro-armed — скрыты.

**Структура:** .map-teaser становится <div>; вся площадь — <a.mt-link> на
guide#map (label+CTA внутри); пины pointer-events:none; карточки — отдельные
<a> поверх (никаких вложенных ссылок). Селекторы HomeIntro (.map-teaser,
.mt-pin) не ломаются.

## Часть 2 — переход тизер → большая карта гида

Воспроизведено на проде 2026-06-12 (playwright):

**Баг А — приземление выше карты.** /guide#map: якорный скролл срабатывает до
загрузки картинок сверху → страница останавливается на ~390px выше (Chrome потом
доводит, Safari — нет). Фикс: инлайн-скрипт на guide (EN+RU): если hash=#map —
после load и в моменты 300/800/1500/2500мс довести скролл до .mapblock
(с учётом scroll-margin); доводка отключается при первом действии юзера.

**Баг Б — карта открыта с неверным центром.** frameFlats = центроид квартир +
жёсткий zoom 15: на узком контейнере половина оранжевых пинов за краем
(скриншот mob-t7). Фикс в rega-map.js: fitBounds по квартирам с padding 56,
после idle зум капится на frameZoom (15) — тесная группа не перезумится,
разлёт квартир корректно вмещается в любой контейнер.
